# plus-smart-home-tech

## Порядок запуска

1. Запустить докер контейнер
- Выполнить команду в папке с проектом: "**docker-compose up -d**"

2. Запустить модуль **infra/discovery-server**
3. Запустить модуль **infra/config-server**
4. Запустить модуль **infra/gateway-server**
5. Запустить модуль **stat/stats-server**
6. Запустить модуль **core/main-service**
